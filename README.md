### Install and usage:
Tested on:
* Python 3.6.8
* pip==21.3.1

Install and run:
```bash
pip install -r requirements.txt
python main.py
```
Libs in requirements.txt:
* PyQt5==5.15.6
* Pillow==8.4.0
* pycryptodome==3.15.0
